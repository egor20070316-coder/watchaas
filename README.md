# Watchaas

Учебный сайт магазина наручных часов.

## Что есть в проекте

- каталог из 10 моделей часов;
- фильтрация по бренду, механизму и цене;
- открытие карточки товара по нажатию на часы;
- отзывы с фильтром и сортировкой по дате;
- ToDoList с сохранением задач в браузере;
- manifest и service worker для установки на телефон.

## Запуск локально

```bash
python -m http.server 4173
```

После запуска сайт открывается по адресу:

```text
http://127.0.0.1:4173/
```

## Публикация

Сайт можно разместить через GitHub Pages: `Settings -> Pages -> Deploy from a branch`, ветка `main`, папка `/root`.
