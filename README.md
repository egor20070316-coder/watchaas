# Watchaas

Watchaas - это мой учебный сайт магазина наручных часов. Я делал его как небольшую витрину: на сайте можно открыть каталог, выбрать модель часов, посмотреть карточку товара, почитать отзывы и вести список задач.

Проект написан без PHP и базы данных, поэтому он нормально запускается на GitHub Pages. Все данные для каталога, отзывов и задач работают на стороне браузера.

## Что есть на сайте

- главная страница с кратким описанием магазина;
- каталог из 10 моделей часов;
- фильтры по бренду, механизму и цене;
- открытие карточки товара при нажатии на часы;
- раздел с отзывами и сортировкой по дате;
- ToDoList, где можно добавлять и удалять задачи;
- manifest и service worker для установки сайта на телефон как приложения;
- кнопка установки приложения из браузера.

## Как запустить

Самый удобный способ - запустить простой локальный сервер в папке проекта:

```bash
python -m http.server 4173
```

После этого сайт откроется по адресу:

```text
http://127.0.0.1:4173/
```

Для обычного просмотра можно открыть `index.html`, но для проверки PWA лучше запускать через сервер.

## Публикация

Сайт подходит для GitHub Pages. В настройках репозитория нужно выбрать:

```text
Settings -> Pages -> Deploy from a branch -> main -> /root
```

После сохранения GitHub выдаёт ссылку на опубликованный сайт.
